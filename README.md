## Serverless functions

### Creating new serverless functions

#### Node.js 8.x runtime
```bash
serverless create --template aws-nodejs --path ./serverless/stock-info-service
cd ./serverless/stock-info-service
nvm use
yarn add --dev serverless-offline
```
